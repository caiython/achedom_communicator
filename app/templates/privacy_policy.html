Políticas de Privacidade

A Política de Privacidade do aplicativo Achedom descreve como o aplicativo coleta, utiliza e protege os dados das ordens de serviço. Ao utilizar o aplicativo, você concorda com os termos desta política.

Para fornecer seus serviços, o aplicativo Achedom realiza uma solicitação na API da plataforma DeskManager para obter informações relevantes das ordens de serviço. Esses dados incluem, mas não se limitam a, códigos de chamados, datas, descrições e prioridades. Essas informações são armazenadas localmente em um banco de dados para possibilitar o uso do aplicativo.
As informações coletadas pelo aplicativo Achedom são utilizadas exclusivamente para o funcionamento adequado do aplicativo. As informações das ordens de serviço obtidas na API da plataforma de chamados são armazenadas localmente para permitir a automatização do processo de comunicação com os operadores por meio do WhatsApp.
O aplicativo Achedom não compartilha informações coletadas dos usuários com terceiros.
Todas as informações coletadas e armazenadas pelo aplicativo Achedom são tratadas de forma segura e protegidas contra acesso não autorizado.
Toda a informação é acessada através do aplicativo web.
A Política de Privacidade do aplicativo Achedom pode ser atualizada ocasionalmente.
